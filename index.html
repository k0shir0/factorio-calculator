Original calculator

The production calculator — the Factory, Visualize, Resources and Settings tabs, the recipe solver, and all of the underlying Factorio game-data handling — was created by Kirk McDonald. It is the result of years of work and is shared under the Apache License 2.0.

Original source: github.com/KirkMcDonald/kirkmcdonald.github.io

This version

Modifications by k0shir0 (github.com/k0shir0):

  • Reskinned to a minimal black & mint-green theme.
  • Added the Blueprints library — a gallery of builds, each with its own article (build video, screenshots, importable blueprint string, and notes), authored through a built-in editor.
  • Added a Ratios tab — a visual reference of common production ratios (steam power, smelting, oil cracking, green circuits) drawn with the game's own icons.
  • Trimmed the data set to base vanilla Factorio 2.0.

Code review & cleanup

Bugs

  1. Default data set pointed at a missing key. DEFAULT_MODIFICATION was "2-0-10", which is not one of the available data sets, so the Settings data-set selector could initialise with no option actually selected (it only worked by falling back to the first item in the list). Pointed at the real key.
  2. Orphaned Debug-tab code. After the Debug tab was removed, debug.js, renderDebug() and the toggleDebug handler were still imported and wired, pointing at page elements (#debug_tab, #render_debug) that no longer exist. Removed.

Code trimmed

≈ 1,071,000 characters (~1.05 MB) of unused files deleted — almost entirely an abandoned Graphviz visualiser that was never enabled:

  • graphvizlib/graphvizlib.wasm — 870,550 bytes (an unused WebAssembly build of Graphviz)
  • graphvizlib/index.min.js — 106,255 bytes
  • d3-graphviz.js — 83,486 bytes
  • boxline.js — 7,919 bytes (a superseded, unreferenced visualiser)
  • debug.js — 2,799 bytes (the removed Debug tab's renderer)

Performance

  • Game data is now cached. The recipe data file (~238 KB) was re-downloaded on every page load (cache: "reload"); it is now cached by the browser (the file name is versioned, so that is safe), making repeat loads much faster.
  • Non-blocking scripts. The library scripts (d3, dagre, BigInteger, pako, popper, marked — ~500 KB total) now load with defer, so the page parses and paints without waiting on them.
  • Lighter load & deploy. One fewer module is fetched on load, and dropping the ~1 MB Graphviz bundle makes the whole site clone and deploy faster.

Sources

References for the data, the production ratios, and build information. Ratio figures were also cross-checked against this site's own vanilla-2.0.55 game data.

  • Factorio Calculator by Kirk McDonald — the calculator this site is built on.
  • Official Factorio Wiki — item, recipe, and entity data (incl. Boiler, Steam engine, Oil refinery).
  • FactorioPrints — source of the community blueprints in the Blueprints tab. Every imported entry names its original author and links back to the original FactorioPrints post.
  • Factorio Cheat Sheet — common production ratios and reference numbers.
  • Factorio Reference Sheet (referencio.info).
